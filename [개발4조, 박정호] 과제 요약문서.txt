#. 액티비티 구성
1) 초기화 페이지(LoadingActivity)
	=> 로그인 및 회원정보 참조를 위한 DB용 객체 생성/초기화 기능 수행

2) 로그인 페이지(SignInActivity)
	=> 사용자의 입력 및 회원정보 참조/대조를 통한 로그인 기능 수행
	=> 이동가능 : (1) 종료, (2) 회원정보 페이지, (3) 회원가입 페이지

3) 회원가입 페이지(SignUpActivity)
	=> 사용자의 입력 및 회원정보 갱신, 로그인 기능 수행
	=> 이동가능 : (1) 로그인 페이지, (2) 자기소개 페이지

4) 자기소개 페이지(HomeActivity)
	=> 인증된 사용자 정보를 통한 자기소개 DB값 참조 및 출력 기능 수행
	=> 이동가능 : (1) 로그인 페이지
